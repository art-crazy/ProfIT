Реализуйте класс Validator, который будет проверять строки.

К примеру, у него будет метод isEmail c параметром, который принимает строку и проверяет, является ли она корректной для адреса электронной почты.

Если является - возвращает true, если не является - то false.

Кроме того, класс будет иметь следующие методы:

метод isDomain для проверки домена,

метод isDate для проверки даты

и метод isPhone для проверки телефона.
